# Hi, I'm Kunwar

I build software across digital ink, backend systems, geospatial data, and applied research. I am interested in tools that make complex information easier to inspect, edit, and trust.

## Currently building

[Neeh SDK](https://github.com/kunwarpadda/Neeh-SDK) is an open-source digital-ink SDK for working with handwriting, sketches, and diagrams as structured data. It focuses on stable stroke identity, temporal and geometric analysis, bounded context retrieval, and validated editing across Python and C++ interfaces.
